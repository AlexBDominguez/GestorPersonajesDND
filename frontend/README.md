## Backend API

The backend API runs on `http://localhost:8081`

### Main endpoints:
- `/api/characters` - Character management
- `/api/races` - Available races
- `/api/classes` - Character classes
- `/api/spells` - Spells catalog

See backend documentation for complete API reference.
